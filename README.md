# Shoot The Fruit
A simple, fun game written with the Python pgzero (Pygame Zero) module.
It is available for installation on Debian-based systems.

### Installation Instructions
##### Linux (Ubuntu, Mint, Debian):
1. Install all dependencies:
```bash
sudo apt install -f python3 python3-pygame python3-pgzero
```
2. Install the package:
```bash
sudo dpkg -i shoot-the-fruit-1.0-all.deb
```
Most importantly, enjoy the game. I am actively improving the codebase, so make sure to stay tuned for newer versions!
